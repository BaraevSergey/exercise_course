1.Как инициализировать репозиторий? Как можно склонировать репозиторий на локальный компьютер?

git init [name] -  для инициализации
git clone [link] - для ссылок

2. В чем разница между локальным и удаленным репозиторием? Как можно привязать локальный репозиторий к удаленному?

Локальный - на компе, удаленный - в сети)) Удалённый может взять любой)

3. Что делаю команды add, commit и push?
add - добавляет изменения в фиксацию
commit - фиксация изменений
push - пушинг в удалённый репо 
4. Как обновить локальный репозиторий из удаленного? Как обновить сразу локальный репозиторий и ветку на которой вы находитесь с удаленного репозитория?
git pull - Для синхронизации с удалённым репо
git fetch - только для ветки
5. Что делают команды reset и revert? Когда можно использовать reset, а когда нужно использовать revert?
reset - откат к коммиту с удалением тех, которые были после откатываемого коммита
revert - помогает восстановить потом откаченные коммиты
6. Что делают команды merge и rebase? В чем разница между ними? Когда рекомендуется использовать merge, а когда rebase?
merge, rebase - берут коммиты из ветки другой в свою
rebase - если только мы работаем с удаленным репо, merge - почти везде)) , так как merge всё в один коммит пихает, а rebase - продолжает ветку кучей коммитов и слиться обратно потом будет почти невозможно
7. Как взять любой коммит из любого места репозитория в текущую ветку?
Если я понял вопрос, то тут ответ cherry-pick) 
Коммиты можно черипикать из ветки в ветку
